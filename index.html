4.更换rehat7的yum源

发表于 2016-03-30

删除redhat原有的yum源

rpm -aq | grep yum | xargs rpm -e –nodeps
rpm -aq | grep python-iniparse | xargs rpm -e –nodeps

下载新的yum安装包

http://mirrors.163.com/centos/7/os/x86_64/Packages/python-iniparse-0.4-9.el7.noarch.rpm
http://mirrors.163.com/centos/7/os/x86_64/Packages/yum-metadata-parser-1.1.4-10.el7.x86_64.rpm
http://mirrors.163.com/centos/7/os/x86_64/Packages/yum-3.4.3-132.el7.centos.0.1.noarch.rpm
http://mirrors.163.com/centos/7/os/x86_64/Packages/yum-plugin-fastestmirror-1.1.31-34.el7.noarch.rpm

安装yum软件包

rpm -ivh python-iniparse-0.4-9.el7.noarch.rpm
rpm -ivh yum-metadata-parser-1.1.4-10.el7.x86_64.rpm
rpm -ivh yum-3.4.3-132.el7.centos.0.1.noarch.rpm yum-plugin-fastestmirror-1.1.31-34.el7.noarch.rpm

注意：最后两个安装包要放在一起同时安装，否则会提示相互依赖，安装失败。

更改yum源 #我们使用网易的CentOS镜像源

cd /etc/yum.repos.d/
wget http://mirrors.163.com/.help/CentOS6-Base-163.repo
sudo gedit CentOS6-Base-163.repo
查找与替换，将 $releasever 换成 7 ，将 几个6 也换成 7 。保存。
把原有的rehat源移到备份文件夹
把网易源CentOS6-Base-163.repo的第一个enabled=0改为enabled=1

清理yum缓存

yum clean all
yum makecache #将服务器上的软件包信息缓存到本地,以提高搜索安装软件的速度
yum install fcitx #测试yum是否可用
